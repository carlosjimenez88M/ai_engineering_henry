# Conceptos base

## Objeto y clase
- Clase: plantilla.
- Objeto: instancia concreta.

Pregunta guía: ¿qué propiedades y comportamientos comparten estos objetos?

## Estado y comportamiento
- Estado: datos internos.
- Comportamiento: métodos que operan sobre esos datos.

Una buena clase protege su estado y expone métodos claros.
