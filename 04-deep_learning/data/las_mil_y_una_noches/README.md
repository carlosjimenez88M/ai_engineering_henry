# Corpus local: Las mil y una noches

Este corpus contiene extractos largos, normalizados y adaptados en espanol a partir de relatos de dominio publico de `Las mil y una noches`.

## Por que esta en el repo

- para que las notebooks de NLP y Transformers funcionen sin internet,
- para trabajar con texto narrativo largo y con vocabulario recurrente,
- para evitar dependencias externas durante la clase.

## Que no es este corpus

- no es una edicion critica,
- no es una transcripcion academica,
- no reemplaza la lectura completa de la obra.

Es un corpus docente curado para ejercicios de tokenizacion, ventanas de contexto, atencion, modelado autoregresivo y generacion local.
